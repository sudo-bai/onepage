Skadi Homepage 浏览器扩展版（本地离线）

用途
- 直接把新起始页项目整体打包进扩展
- 覆盖 Chrome / Chromium / Brave 的新标签页
- 不依赖线上站点
- 保留项目原有的本地数据、导出备份、恢复备份功能

目录结构
- manifest.json      扩展清单
- app/               打包后的主页应用
- options.html       简单说明页
- style.css          说明页样式

当前能力
- 已支持：新标签页替换为本地版主页
- 已支持：导入旧备份 JSON
- 已支持：之后完全离线使用本地数据
- 已移除：自定义网站地址配置逻辑

安装方法
1. 打开 brave://extensions/ 或 chrome://extensions/
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录：/home/skadi/文档/hermes/homepage-extension-project
5. 打开新标签页，确认页面已变成你的主页

说明
- 这个扩展现在不再加载外部自定义网站，只负责把你的主页项目作为新标签页打开
- 浏览器地址栏会显示浏览器自己的新标签页/扩展页环境，而不是跳到外部网址
- 扩展内数据默认不自动同步，这符合“脱离云端”的目标
