Onepage Extension

Onepage - 自定义新标签页

一个简洁美观的浏览器新标签页扩展，提供快速导航、搜索引擎切换和个性化快捷方式。

打开主页

主要特性

  • 🚀 快速启动 - 替换浏览器默认新标签页
  • 🎨 美观界面 - 现代化的深色主题
  • 🔍 智能搜索 - 支持多搜索引擎快速切换
  • 📍 快捷导航 - 可自定义的网站快捷方式
  • 💾 数据备份 - 支持导出和恢复配置
  • 🖼️ 随机背景 - 智能缓存，节省流量

安装方式

  • 从 GitHub Releases 下载最新版本
  • 解压 ZIP 文件到本地目录
  • 在浏览器扩展页面开启"开发者模式"
  • 点击"加载已解压的扩展程序"
  • 选择解压后的目录
